# kursovaia-rabota
## Анализ математической функции (C)
Данная курсовая работа, выполненная по дисциплине «Основы программирования и алгоритмизации», представляет собой консольное приложение на языке C для анализа математической функции. Программа обеспечивает проведение основных исследований с обязательной проверкой входных данных и области определения функции.
## Постановка задачи
- Вычисление значения заданной функции в произвольной точке.
- Построение таблицы значений функции на указанном интервале.
- Поиск экстремума (минимума/максимума) функции на заданном отрезке
- Нахождение аргумента по известному значению функции.
- Численное вычисление производной функции в точке.
- Добавление возможности записи значений в файл и чтение значений из файла для построения таблицы

Программа должна быть оснащена консольным меню для навигации, а также реализовывать проверку вводимых данных на корректность и контроль принадлежности точек области определения функции.
## Математическая модель
Функция задана кусочно. При x < 1 значение функции вычисляется по формуле f(x)= e^|x| / arctan(1+x^2). При 1 <= x <= 6 значение функции определяется, как f(x)= x^2 * cos(3x) * tanh(x) / √1+x^6. При x >=6 значение функции вычисляется по формуле f(x) = Σ (-1)^n x^2n+1 / (2n+1)!! * (n+3), где n изменяется от 0 до 4.
## Общий алгоритм работы программы
Программа реализует вычисление кусочно заданной функции, построение таблицы значений, поиск экстремумов, численное дифференцирование, а также ввод и вывод данных в файл. Управление осуществляется через текстовое меню. Все вычисления выполняются с использованием стандартных математических функций языка C.
# Описание ключевых функций
### 1.	double f(double x) 
Вычисляет значение заданной кусочной математической функции f(x) в точке x в соответствии с условиями задачи.
#### Принимаемые параметры:
- x - аргумент функции, вещественное число
#### Возвращаемое значение:
- значение функции f(x)
### 2.	double solve(double y, double a, double b, double eps)
Осуществляет поиск значения аргумента x, при котором значение функции f(x) приближённо равно заданному значению y на интервале [a,b].
#### Принимаемые параметры:
- y- заданное значение функции f(x)
- a – начало интервала поиска
- b – конец интервала поиска
- eps – допустимая погрешность
#### Возвращаемое значение:
- Найденное значение x, при котором f(x)  ≈ y, если значение не найдено – возвращается NaN.
### 3.	double derivative( double x)
Вычисляет приближённое значение первой производной функции f`(x) в точке x численным методом.
#### Принимаемые параметры:
- x – точка, в которой вычисляется производная
#### Возвращаемое значение:
- Приближенное значение производной f`(x).
### 4.	void table(double a, double b, double step) 
Формирует и выводит на экран таблицу значений функции f`(x) на интервале [a,b] с заданным шагом.
#### Принимаемые параметры:
- a – начало интервала
- b – конец интервала
- step – шаг изменения аргумента
#### Возвращаемое значение:
- функция ничего не возвращает (void).
### 5.	void minmax(double a, double b, double step, double* minx, double* minv, double* maxx, double* maxv) 
Осуществляет поиск минимума и максимума функции f(x) на заданном интервале [a,b] численным методом
#### Принимаемые параметры:
- a – начало интервала
- b – конец интервала 
- step – шаг перебора аргумента
- minx – указатель на x, при котором достигается минимум
- minv – указатель на минимальное значение функции
- maxx – указатель на x, при котором достигается максимум
- maxv – указатель на максимальное значение функции
#### Возвращаемое значение:
- функция ничего не возвращает
### 6.	void table to file(double a, double b, double step, const char* filename)
Функция table_to_file выполняет запись таблицы значений математической функции f(x) в текстовый файл. Она последовательно вычисляет значения функции на заданном интервале с определенным шагом и сохраняет результаты в файл в формате “x f(x)” для дальнейшего использования или анализа.
#### Принимаемые параметры:
- a - начальная точка интервала
- b - конечная точка интервала
- step - задает шаг изменения аргумента x
- filename - задает имя файла для записи данных
#### Возвращаемое значение:
- Функция не возвращает значение
### 7.	void table_from_file(const char* filename)
Функция table_from_file выполняет чтение и отображение таблицы значений функции f(x) из текстового файла. Она читает заранее сохраненные данные из указанного файла и форматировано выводит их на экран в виде таблицы.
#### Принимаемые параметры:
- filename- задает имя файла для чтения данных
#### Возвращаемое значение:
- Функция не возвращает значение

# Формат входных и выходных данных
В качестве входных данных программа использует вещественные числа с двойной точностью (double) для задания аргумента, границ интервалов и значений функции. Выбор пунктов меню и количество точек табуляции задаются целыми числами типа int. Результаты работы (числовые значения и таблицы) выводятся в консоль в отформатированном виде.
